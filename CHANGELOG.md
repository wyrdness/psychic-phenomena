# Changelog

All notable changes to this repository will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-14

### Added
- Initial repository structure
- Comprehensive README.md with phenomenon documentation
- api.json following Wyrdness schema v1.0.0
- SOURCES.md bibliography
- Issue templates for experience reports, corrections, and source additions
- Pull request template

### Data Included
- Basic phenomenon metadata and classification (PSYCHIC_PHENOMENA)
- 7 types of psychic abilities (Telepathy, Clairvoyance, Precognition, Psychokinesis, Mediumship, Remote Viewing, Retrocognition)
- Famous experiments (Rhine ESP cards, Ganzfeld, CIA Stargate, PEAR, Presentiment)
- Famous psychics (Edgar Cayce, Uri Geller, Ingo Swann, Pat Price, Joe McMoneagle)
- Evidence catalog and methodology
- Theories (paranormal, scientific, skeptical)
- Replication crisis documentation
- Research organizations (Rhine Research Center, Parapsychological Association, etc.)
- Cultural significance
